# 开发者指南

面向 **MindIE Motor 代码贡献者**，涵盖软件实现细节、类图、数据流与开发工具。

## 内容侧重

- 组件内部实现（Controller、Coordinator、Engine Server、Node Manager）
- 类图、数据流、模块交互
- 构建工具（文档构建、镜像制作）

## 与设计文档的区别

| 开发者指南 | 设计文档 |
|-----------|---------|
| 软件实现细节 | 特性原理与设计说明 |
| 面向代码贡献者 | 面向深度用户、架构师 |
| 侧重 "代码怎么写出来的" | 侧重 "为什么这么设计" |

如果你需要理解特性的设计原理与架构决策，请查看 [设计文档](../design/README.md)。
